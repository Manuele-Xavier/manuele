Moonvier

Moonvier é um projeto acadêmico desenvolvido com o intuito de ser um e-commerce de roupas unissex.

Foi desenvolvido apenas o front-end do site, e as tecnologias utilizadas foram: HTML5, CSS3 e JavaScript.

Coorgânicas

Coorgânicas é uma plataforma desenvolvida no curso Code-XP da escola SENAI Informática. Proposto por uma empresa real e consiste na ligação de agricultores familiares á mulheres de baixa renda, os agricultores fornecem alimentos orgânicos com preço acessível, para que as mulheres possam fazer marmitas gourmet gerando uma renda extra.

O site tem seu front-end em HTML5, CSS3 e ReactJs, a linguagem de programação utilizada para o backend foi o C# e o banco de dados foi realizado em SQLServer.
